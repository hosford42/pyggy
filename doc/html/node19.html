4.1 The Code section

The first section type is the code section. It specifies code to be copied directly to the output file (unindented). The code section is started with the word code followed by a colon and a newline. All the indented code that follows is copied directly to the output file. For example: 

code :
    def printjoined(l) :
        print " ".join(l)

    printjoined["this","is","a","test"])

If more than one code section appears in the spec file, the code from each section is gathered together and copied out to the output file.
